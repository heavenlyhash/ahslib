The AHS library has three primary goals: remove common drudgery from coding in java wherever possible, make writing multithreaded applications easier, and codify common design patterns so that good code becomes more self-documenting. The AHS library is free and open source, currently licensed under cc:by-nc-nd-3.0. See http://creativecommons.org/licenses/by-nc-nd/3.0/ for a brief explanation. Note the no-derivs clause: if you wish to alter or expand the library, get in touch and we'll get you write perms to the repo. The subversion repository lives at https://rubicon.exultant.us/svn/ahs/. Documentation for new users is available here.
