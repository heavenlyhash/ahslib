The AHS library has three primary goals: remove common drudgery from coding in java wherever possible, make writing multithreaded applications easier, and codify common design patterns so that good code becomes more self-documenting.

The AHS library is free and open source, currently licensed under cc:by-nc-nd-3.0. See http://creativecommons.org/licenses/by-nc-nd/3.0/ for a brief explanation. Note the no-derivs clause: if you wish to alter or expand the library, get in touch and we'll get you write perms to the repo. I'd much rather you send feedback and patches to back to the central community, instead of fracturing improvements up into unmerged private versions. (Using this library, extending its classes, and implementing its interfaces does not count as creating derivitive work as forbidden by the license.)

The subversion repository lives at https://rubicon.exultant.us/svn/ahs/.

Documentation for new users is available here. (Also, this documentation is in the repository itself, so you can both read and contribute to it that way.)
